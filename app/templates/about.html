About

This site was created by Sam Friedman.

The goal is to make the website easy enough to use for novice satellite watchers but detailed enough for experts. It is in active development, so check back for updates and new features.

Orbit and satellite data is gathered from Celestrak three times a day. Coordinate frame transformations use the IAU FK5/1976 transformations and WGS84 Earth constants.

For detailed description of the technology stack and astrodynamics computation, see my personal blog. The code for the site is open source and can be viewed at github.com/samtx/passpredict-app.
