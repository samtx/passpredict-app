About

This site was created by Sam Friedman.

It is built with a Python backend using FastAPI and a frontend written in Svelte, all hosted on a Digital Ocean virtual private server. It uses Redis for caching API calls and intermediate orbit propagation values. Orbit and satellite data is gathered from Celestrak and stored in a Sqlite database.

The pass prediction algorithm uses python-sgp4 for orbit propagation and custom C functions based on IAU SOFA library for the coordinate frame rotations and timescales.
