About

This site was created by Sam Friedman.


It is built with a Python backend using Starlette and FastAPI. The frontend is server-rendered with Jinja templates. It uses Postgresql and Redis for data storage and caching. Caddy server is the reverse proxy for the Starlette application and serves the static files. All of the applications are hosted on a Digital Ocean virtual private server.


Orbit and satellite data is gathered from Celestrak three times a day. Orbit propagation and overpass prediction is computed with the Orbit Predictor python library. It uses the SGP4 propagator python-sgp4, written by Brandon Rhodes, with WGS84 Earth constants. Coordinate frame transformations use the IAU FK5/1976 coordinate frames. Cython is used to speed up some functions.
